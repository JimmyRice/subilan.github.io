# 如何在本站上添加友链？

您可以通过这里的联系方式联系到我，或者直接在 GitHub 页面提 Issue，标明「友链申请」。您甚至可以直接提交 Pull Request，仅需依照 /.vuepress/components/FriendLinks.vue 内的写法继续在该文件中添加内容即可。在提交之前请参考如下规则。目前友链申请是开放的。

  • 开启全站 HTTPS
  • 非免费二级域名
    • 判断标准：此二级域名和顶级域名均归您所有即可。
  • 个人博客性质

如果您觉得您已经满足了上述要求，可以参考如下格式提交申请：

网站名称：（必填）

网站介绍：（必填）

网站代表色：（选填）

网站地址：（必填）

头像（圆/方皆可）：（必填）

由于工作原因，我会在节假日给您答复。
