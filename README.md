## 博客的运作方式...？

自 2019/8/31 起，本博客不再使用 GitHub Webhooks 进行部署，反之直接使用 GitHub Pages。本博客的所有构建后产生的静态文件将会被部署到 Branch `gh-pages`。

同时，本项目使用 Travis CI 进行自动部署（`vuepress build .`）。
