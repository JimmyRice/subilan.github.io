## 如何在本站上添加友链？

> 直接在博客的 GitHub 上 Pull Request 是不存在的...

您可以通过[这里](/contact.html)的联系方式联系到我，或者直接在 GitHub 页面提 Issue，标明「友链申请」。如果我看到了并且符合了下面的规定，您的友链将会由我本人添加在这里。目前友链申请是开放的。

规定也很简单，没有硬性要求：

- 开启全站 HTTPS
- 非二级域名
- 个人博客性质

如果您觉得您已经满足了上述要求，就请告诉我吧！
